# Initial Concept
The Engineer Career Development Framework (ECDF) application manages employee career progression through assessments, evidence submission, and personal development plans based on an 8-pillar framework.

# Product Guide

## Target Users
- **Software Engineers**: Employees submitting evidence of their work and tracking their career progression.

## Core Value
- **Standardized Progression**: Provide clear, objective criteria for promotions using the 8-pillar framework.

## Key Features
- **Immutable Evidence Tracking**: Capture evidence submissions as immutable facts with impact, complexity, and personal contribution.

## UX Focus
- **Premium Dark Mode Aesthetic**: Use glassmorphism and modern typography for a sleek internal tool.
